# フル板 訓練用ダミーアプリ

株のフル板の値動きを見る訓練（目慣らし）のためのダミーアプリです。
数字はすべてランダム生成の架空データで、実在の銘柄・相場とは関係ありません。

## 使い方

ブラウザで以下を開くだけです。

https://kumakuma-git.github.io/fullita-training/

iPadの場合は Safari で開き、共有ボタン →「ホーム画面に追加」でアプリのように使えます。

## 元ファイル

`C:\Git\investment\stocks\フル板訓練\フル板ダミー.html`（プライベートリポジトリ側で管理）
を `index.html` としてコピーしたものです。修正はそちらで行い、ここへコピーしてプッシュすると反映されます。
